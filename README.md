# JavaScript Projects Collection

This collection of projects is built using core JavaScript concepts, showcasing a range from beginner-friendly applications to intermediate-level systems. Each project focuses on improving real-world coding skills by applying JavaScript to solve interactive problems, manipulate the DOM, manage data dynamically, and enhance user experiences.

---

## Projects

| Project | Description | Concepts Used |
|---|---|---|
| **Currency Converter** | Real-time currency conversion using exchange rate API | Fetch API, Async/Await, DOM |
| **City Stats** | Displays live city data and statistics dynamically | API integration, DOM Manipulation |
| **Memory Game** | Card-matching game with score tracking | DOM events, Arrays, Game logic |
| **Snake Game** | Classic Snake game built in the browser | Canvas API, Keyboard events, OOP |
| **FD Rate Calculator** | Fixed deposit interest calculator with dynamic output | Math operations, DOM, Forms |
| **Digital Clock** | Live clock updating every second | Date object, setInterval |
| **Color Flipper** | Random background color generator | DOM manipulation, Events |
| **Todo App** | Task manager with add/delete functionality | CRUD, Local state management |
| **Navbar** | Responsive navigation bar component | CSS + JS, Responsive design |
| **Async Demo** | Demonstrates asynchronous JS concepts | Promises, Async/Await, Callbacks |
| **OOP Demo** | Object-oriented programming patterns in JS | Classes, Inheritance, Encapsulation |
| **Objects** | Deep dive into JS object manipulation | Objects, Methods, Prototypes |
| **Promise Async Await** | Comparing callbacks vs promises vs async/await | Promise chaining, Error handling |
| **Set and Map** | Usage of ES6 Set and Map data structures | ES6, Data structures |

---

## Concepts Covered

- **DOM Manipulation** — Dynamically updating HTML elements and responding to user events
- **Async JavaScript** — Fetch API, Promises, Async/Await, and callback patterns
- **Object-Oriented Programming** — Classes, constructors, inheritance, and encapsulation
- **ES6+ Features** — Arrow functions, destructuring, template literals, Set, Map, spread operator
- **API Integration** — Fetching and displaying live data from external APIs
- **Event Handling** — Click, keydown, input, and form events
- **Data Management** — Managing application state dynamically without a framework

## How to Run

Each project is self-contained in its own folder. To run any project:

1. Clone the repository
```bash
   git clone https://github.com/vagulavarshini0320/javascript-.git
```
2. Navigate to any project folder
```bash
   cd "Memory Game"
```
3. Open `index.html` in your browser
